# Kopico-Chat

This is a full-stack web application powered by:

- **Frontend**: React 18 + Vite + Tailwind CSS + Zustand + React Router + Socket.IO
- **Backend**: Node.js + Express + MongoDB + Passport (Google OAuth2) + Cloudinary + JWT + Socket.IO
